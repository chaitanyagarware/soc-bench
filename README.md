# SOC-Bench v0

SOC-Bench is a reproducible evaluation framework for LLM-based Security Operations Center (SOC) threat classification systems.

This repository accompanies the paper:

**When the Ruler Is Broken: Parsing-Induced Suppression in LLM-Based Security Log Evaluation**

## Features

- 13-category SOC threat taxonomy
- Fuzzy parsing for robust evaluation
- Parser-agnostic scoring
- Broad-category normalization
- Reproducible evaluation pipeline
